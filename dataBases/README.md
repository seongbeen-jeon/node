# 데이터 베이스 사용하기

RDBMS(MySQL)과 NoSQL(MongoDB)를 사용하는 법을 공부하고 ORM과 ODM을 통해 express App과 연결하는 법을 알아보도록 하겠습니다.

SQL은 다른 다른곳에서 공부 하도록하고 ORM과 ODM을 다루는것을 위주로 알아보겠습니다.

# RDBMS

RDBMS는 관계형 데이터베이스를 관리하는 프로그램입니다. mySQL, Oracle, MSSQL 등의 대표적인 프로그램이 있습니다.

RDB는 테이블간 관계를 통해 복잡하고 거대한 테이블을 만들어낼 수 있습니다.

# NoSQL

NoSQL은 Not Only SQL의 약자로 통하며 테이블간의 관계를 지원하지 않으며 테이블의 구조 또한 명시적으로 선언할 필요가 없어 더욱 자유로운 사용이 가능합니다. 이는 장점이자 단점이 될 수 있습니다.

MongoDB를 통해 NoSQL을 공부할 것입니다. 몽고디비는 Js의 객체를 사용하기 때문에 접근성이 좋으며 노드 서버에서 주로 사용되는 DB라고 합니다.
